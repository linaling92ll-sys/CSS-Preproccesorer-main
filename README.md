# Mitt CSS-Preproccesorer projekt

Det här är en webbplats som själva färgläggningen och allt sådant är skriven
med SASS-kod. Jag har byggt vidare på hemsidan genom att lägga till olika animationer
på en egen undersida.

## Det som är nytt på webbplatsen
Jag har lagt till två nya undersidor på webbplatsen.
En är en kartfunktion och den andra är där jag visar ett stapeldiagram och ett cirkeldiagram
som visar de mest sökta kurserna och de mest sökta programmen på Mittuniversitetet.

## Kunna se dokumentationen
För att kunna se dokumentationen så kan man köra i terminalen på projektet
npx jsdoc src -r -d docs
